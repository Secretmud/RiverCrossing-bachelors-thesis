Some usefull linux stuff

Symlinks

Working on files using a symbolic link:
run the following in your terminal of choice:
ln -s $(pwd)/{dir to link} /{where to link it}

If you are met with a 403 error page. run the following:
chmod o+x /home/username

Samba

Run the following to mount a samba share while you're on the correct network.
sudo mount -t cifs //student.cs.hioa.no/{username} /mnt/student.cs.hioa/ --verbose -o username={username},workgroup=ada
